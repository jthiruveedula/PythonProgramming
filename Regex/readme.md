## Regex Cheet Sheet

**Python Regex patterns**
 - Sample codes attached

```
[] --> Square Brackets
. --> period
^ --> caret
$ --> dollar
* --> asterisk
+ --> plus
? --> QuestionMark
{} --> Braces
() --> group
\ --> Backslash
| --> pipe
```
